# Art-Ist Game
A virtual museum where you can view two different galleries: Harvard Museum and Art Institute of Chicago, and play the art-artist matching game

To retrieve painting information, [Harvard Museum](https://github.com/harvardartmuseums/api-docs) and [Art Institute of Chicago](https://api.artic.edu/docs/#introduction)'s third party APIs are used.

For authentication, Firebase's sign-in with email and password method is used.
